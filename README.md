# Prepaid Meter for Energy Management & Overvoltage Protection

# Description

Developed a smart prepaid energy meter system to monitor electricity usage and protect appliances from overvoltage. The system enables users to recharge in advance and track real-time consumption through a display interface.

# Key Features
Real-time energy monitoring using Arduino and LCD display
Overvoltage protection using voltage regulators and transformers
Buzzer alerts for abnormal voltage conditions and low balance
Prepaid recharge system for controlled electricity usage
Basic predictive approach to optimize power consumption

# Technologies Used

Arduino, Embedded C, LCD Display, Voltage Regulators, Transformers, Buzzer

# Working

The system tracks energy consumption and displays it on an LCD screen. Users recharge the meter, and usage is deducted accordingly. When voltage exceeds safe limits, protection mechanisms are triggered and alerts are generated using a buzzer. The system also helps users manage consumption more efficiently.

# Applications

Residential energy management, prepaid billing systems, power monitoring and protection
